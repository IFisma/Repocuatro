1. HTML – La estructura 📚 Doc oficial en DevDocs: HTML Qué aprender primero: Etiquetas básicas: html, head, body, h1 a h6, p, a, img, div, span. Formularios: form, input, label, textarea, button, etc. Listas: ul, ol, li. Tablas: table, thead, tbody, tr, td. 📌 Leé secciones como "Element", "Global Attributes" y "Forms".

2. CSS – El estilo 📚 Doc en DevDocs: CSS Temas clave para empezar: Selectores: .clase, #id, elemento, combinadores. Propiedades básicas: color, background, font-size, margin, padding, border. Posicionamiento: display, position, flex, grid. Pseudoclases: :hover, :first-child, :nth-child(). Transiciones simples: transition, transform. 📌 En DevDocs buscá "CSS Properties" para encontrar cada propiedad con ejemplos.

3. JavaScript – La interacción 📚 Doc en DevDocs: JavaScript Comenzá por: Variables: let, const. Tipos de datos: strings, números, booleanos, arrays, objetos. Condicionales: if, else, switch. Bucles: for, while, forEach. Funciones: function, arrow functions. DOM: document.querySelector, addEventListener, manipulación de HTML. 📌 Buscá la sección “Standard built-in objects” y “DOM” en DevDocs para explorar el API.

4. DOM (Document Object Model) 📚 Doc en DevDocs: DOM Aprendé a: Modificar elementos (innerHTML, classList, etc.). Capturar eventos (click, input, submit). Crear elementos (createElement, appendChild). 📌 La práctica con el DOM es clave para hacer páginas interactivas.

5. Herramientas útiles Además podés revisar: Git (para control de versiones). Console (para usar la consola del navegador). Web APIs (para cosas como fetch, localStorage, etc.). ✅ Consejo de estudio Elegí una sección por semana. Leé y luego hacé mini-proyectos (ej: una galería, una lista interactiva, un formulario). Usá la consola del navegador para experimentar.
